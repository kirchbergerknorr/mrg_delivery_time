** USAGE
Symmetrics_DeliveryTime creates a new attribute
"delivery_time" for the products.

** FUNCTIONALITY
*** A: Creates a new product attribute "delivery_time" with
        a default value "2-3 days"
*** B: Creates a new column "delivery_time" in the product grid
        with the filtering option
